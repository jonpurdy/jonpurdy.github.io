P*M Framework

P*M framework is intended to be a collection of reference documents and tutorials about how I manage software projects and products. The asterisk acts as a wildcard since aspects of project, product, and program management are covered.

My philosophy is based on Agile Scrum and Kanban principals mixed with my practical experience building products, as well as automation and remote work techniques. I also cover topics related to remote work, such as optimizing wifi configuration and microphone setup for videoconferencing.
